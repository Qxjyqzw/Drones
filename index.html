Abstract

The project is a 3D competitive game named Drones. The main mechanism of the game is that players control the drones to pass level while fighting against the enemies. The project is mainly developed based on Three.js and Cannon-es.js. The biggest highlights include fascinating 3D graphics, realistic physics engine, immersive UI design, and high-difficulty operation experience.
